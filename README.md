# 🚀 Chat Application Backend (FastAPI)

A real-time chat backend built using FastAPI, PostgreSQL, and WebSockets.  
Supports private messaging, group chat, authentication, and message persistence.

---

## 📌 Features

### 🔐 Authentication
- JWT-based login system
- Secure API access using OAuth2

### 💬 Messaging
- Private 1:1 chat
- Group chat (rooms)
- Real-time messaging using WebSockets

### 🏠 Rooms
- Create chat rooms
- Join rooms
- Multi-user group messaging

### 💾 Persistence
- Messages stored in PostgreSQL
- Chat history APIs

---

## 🛠️ Tech Stack

- **Backend:** FastAPI
- **Database:** PostgreSQL
- **ORM:** SQLAlchemy
- **Auth:** JWT (python-jose)
- **WebSocket:** FastAPI WebSockets
